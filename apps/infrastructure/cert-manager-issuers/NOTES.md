# cert-manager-issuers

## Type

Plain Kustomize app.

## Purpose

Bootstraps the cluster's internal CA for TLS certificate issuance. Creates:

1. `selfsigned-bootstrap` — a self-signed ClusterIssuer used only to generate the root CA certificate
2. `lab-ca` Certificate — a self-signed root CA stored in the `lab-ca-key-pair` secret in the `cert-manager` namespace
3. `lab-ca` ClusterIssuer — the default issuer for all internal `*.lab.bartos.media` services

## Hostnames

None — this app provides cluster-wide issuer policy only.

## Secrets

- `lab-ca-key-pair` (namespace: `cert-manager`) — auto-generated by cert-manager, contains the CA key pair. **Not stored in Git.**

## Storage

No persistent storage required.

## Notes

- Sync wave `0` — deploys after cert-manager controller (wave `-1`) is ready
- The CA certificate has a 10-year duration with 1-year renewal window
- To add public issuers (e.g. Let's Encrypt), add new ClusterIssuer manifests here without removing `lab-ca`
